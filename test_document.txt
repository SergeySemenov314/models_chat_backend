Это тестовый документ для проверки RAG системы.

В этом документе содержится информация о тестировании системы Retrieval-Augmented Generation.

RAG система позволяет находить релевантную информацию из загруженных документов и использовать её для ответов на вопросы пользователей.

Основные компоненты системы:
1. Извлечение текста из документов
2. Разбиение на чанки
3. Генерация эмбеддингов
4. Векторный поиск
5. Интеграция найденной информации в ответы модели

Система поддерживает форматы: PDF, DOCX, TXT.

Для тестирования используйте флаг useRag: true в запросе к API.
